Process Documents
---
POST /webhooks/process-documents

example payload
```json
{
    "case_id": "acd1b94048d541f8a02eef43",
    "webhook": "https://example.com",
    "is_override": true
}
```

example response 200:
```json
{
    "data": {},
    "err": false,
    "err_code": -1,
    "msg": "Accepted"
}
```
